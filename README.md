# Claude Leak Details Explorer

This repository contains a Next.js documentation explorer for browsing the copied `claude/src` tree with generated summaries, file metadata, and source previews.

## App Location

The application lives in [`my-app`](./my-app).

## Local Setup

```bash
cd my-app
npm install
npm run dev
```

Open `http://localhost:3000` to view the explorer.

## Useful Commands

```bash
cd my-app
npm run dev
npm run build
npm run lint
npm run docs:manifest
```

The production build regenerates the docs manifest automatically before compiling.
